# CMPSC463-Project1
To use this program, you’ll need to install several libraries in Python, including pandas, numpy, and matplotlib. 
You can install these libraries by entering pip install “library name here” into your virtual environment console. 
To run the program, you need to download the dataset (https://www.kaggle.com/datasets/anseldsouza/water-pump-rul-predictive-maintenance) and put the .csv file into the folder with the .py file.
Then, you must download the project from GitHub and extract it (which is assumed to be done if you are reading the README file), then open the .py file in a Python interpreter or enter python MatthewWybranski_CMPSC463_Project_1.py into your terminal/command prompt.
